dgame
=====

## Introduction ##

A game engine library for Dart.

## Getting Started ##

1\. Add the following to your project's **pubspec.yaml** and run
```pub install```.

```yaml
dependencies:
  dgame:
  	git: https://github.com/ejsmith/dgame.git
```

2\. Add the correct import for your project.

```dart
import 'package:dgame/dgame.dart';
```
